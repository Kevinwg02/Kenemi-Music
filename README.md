<h1>Kenemi-Music</h1>

<h2>Summary (EN)</h2>
<p>
KenemiMusic is a modern, smooth and fully offline audio player.  
It automatically scans music files stored on the device and organizes your library into songs, albums,
artists, playlists and folders. The app is completely ad-free and collects no data.
</p>
